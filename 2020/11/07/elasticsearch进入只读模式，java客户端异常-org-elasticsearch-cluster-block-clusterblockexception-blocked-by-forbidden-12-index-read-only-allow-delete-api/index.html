原因

日志里已经很明显了，elasticsearch 进入了只读模式，不能执行删除操作。

Elasticsearch进入只读模式是因为磁盘空间不足造成的，磁盘可用空间过低后（默认低于5%），elasticsearch会进入只读模式，即使增加了磁盘可用空间也不会自动恢复，需要通过如下 curl 命令恢复。
